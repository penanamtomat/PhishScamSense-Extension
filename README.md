# PhishScamSense Extension

Real-time phishing and scam URL detection for Chrome and Firefox.

## Install

Go to the [**Releases**](../../releases) page and follow the instructions for your browser.

### Chrome / Edge / Brave / Opera
1. Download `phishscamsense-*-chrome.zip`
2. Unzip the file
3. Go to `chrome://extensions` → enable **Developer mode**
4. Click **Load unpacked** → select the unzipped folder

### Firefox
1. Download `phishscamsense-*-firefox.zip`
2. Go to `about:addons` → ⚙️ → **Install Add-on From File...**
3. Select the zip file
